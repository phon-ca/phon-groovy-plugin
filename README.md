# Groovy Language plug-in for Phon

This plug-in provides the opgraph node for including
groovy scripts within [PhonShell](https://github.com/phon-ca/phon-shell-plugin).


## Building

After downloading the source,  type

```
mvn package
```

This plug-in is provided with the release version of [Phon](https://www.phon.ca).
